# Android debloat method

## Dependencias

Primeiro precisa instalar o adb via home brew

```sh
brew install android-platform-tools
```

depois o uad-ng via github no repo [UAD-NG](https://github.com/Universal-Debloater-Alliance/universal-android-debloater-next-generation/)

## Ativar depuraçao

Para ativar a depuraçao voce precisa clickar 7 vezes no numero de compilaçao.
Em seguida, ligue a depuraçao sem fio e digite no terminal como no exemplo:

```sh
adb pair 192.168.1.109:42061
```

com o ip e porta que aparecer ao solicitar no menu de depuracao sem fio.
